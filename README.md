# 💼 BKS Quotation System

A single-page web application for creating and managing **Quotations (QT)** and **Investment Proposals (INV)** — built with HTML, Bootstrap 5, and vanilla JavaScript.

🔗 **Live Website:** [https://sutee-bks.github.io/Finance.github.io/](https://sutee-bks.github.io/Finance.github.io/)

---

## 🚀 How to Run

### Option 1: Use GitHub Pages (Already Live)

The website is deployed automatically via GitHub Pages. Just open the link above — no installation needed.

### Option 2: Run Locally

1. **Clone the repository:**
   ```bash
      git clone https://github.com/Sutee-bks/Finance.github.io.git
         cd Finance.github.io
            ```

            2. **Open in browser:**
               - Simply double-click `index.html`, OR
                  - Open with VS Code Live Server extension for best results

                  > ⚠️ No build tools, no npm, no server required — it's a pure HTML file.

                  ---

                  ## 🛠️ Tech Stack

                  | Technology | Usage |
                  |---|---|
                  | HTML5 | Page structure |
                  | CSS3 / Bootstrap 5.3 | Styling & layout |
                  | Font Awesome 6.5 | Icons |
                  | JavaScript (Vanilla) | Logic & interactivity |

                  > All dependencies are loaded via CDN — internet connection required.

                  ---

                  ## ✨ Features

                  - Create **Quotation (QT)** or **Investment Proposal (INV)** documents
                  - Add / edit / delete line items dynamically
                  - Auto-calculate subtotals, discounts, VAT, and grand total
                  - BKS brand color theme (Navy & Blue)
                  - Responsive design (mobile-friendly)
                  - Print / export-ready layout
